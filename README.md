Simple app created to study, still under construction (more features to come)

App create to manage tasks and to get current weather using OpenWeatherMap api.
Stack:
* Retrofit
* Room
* Flow
* Coroutines
* MVVM
* Clean Arch
* Hilt
* Unit tests
* Firebase crashlytics
